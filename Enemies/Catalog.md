# Enemy Catalog

Every combat-ready enemy/NPC sheet found in the source library, extracted for table use.

**Skill Bases** = STAT + Skill (+ mods). Roll **Skill Base + 1d10**. Do not add STAT again.

| Tier | Count vs Edgerunners |
|------|----------------------|
| Mook | 1 per PC (2:1 if combat-heavy) |
| Lieutenant | 1 per 2 PCs |
| Mini-Boss | 1 per 3 PCs |
| Boss | Full fight |
| Hardened * | Same ratios, Hardened Crew only |

Hardened rules: [Homebrew/GM Rules.md](../Homebrew/GM%20Rules.md#what-is-hardened)
