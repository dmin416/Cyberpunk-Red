# Assets NOTES

## Folder layout

- `Bordered/` - ready-to-use bordered tokens (NPC rings), including `Hope Reborn - …` tokens
- `Unbordered/` - handouts / portraits without token rings
- `Blanks/` - blank / Guard placeholders
- `Icons/` - decorative suit marks (diamond, star, heart, moon)
- `Scenery/` - Loke scenery props (downloaded)
- `Scenery Roll20/` - scenery from Roll20 export
- `Vehicles/` - cars, bikes, wrecks, etc.
- `Gridded/` - battle maps with grid
- `Grid free/` - maps without grid
- `Videos/` - animated map / ambience videos

## Named NPCs

See `NAMED NPC INDEX.md` for every named NPC token/handout in Assets.

## Character sorting notes

- **Guard** goes in `Blanks/`
- **Blank** is a named NPC: `Bordered/Hope Reborn - Blank.png`
- **Green Clover** is a guy: `Bordered/Hope Reborn - Green Clover.png`
- Hope Reborn token pack lives in `Bordered/` with `Hope Reborn -` prefix
- Decorative suit marks stay in `Icons/`

## Maps

- `Gridded/` and `Grid free/` stay flat (no Chase / Streets subfolders)
- Filename cues: `GRIDDED` / `-Grid-` -> Gridded; `Gridless` / Panda maps -> Grid free (Panda is gridless even when labeled 20x20)
