# FBC

Full Body Conversion bodies from *Interface RED Vol. 3 - Going Metal* (IR3 p.95-104). Component breakdown for fabrication and sourcing. **10 bodies** - Wiseman (netrunner), Aquarius (diver), and Kildare (medtech) omitted.

## Pricing rules

| Column | Rule |
|--------|------|
| **Cost** | Book / retail (eb) |
| **Fab** | Maker mat cost (one tier below retail; Super Luxury = half). Mats **> 250eb** sourced at **80%** of base mat cost. Mats **<= 250eb** at full fab cost. **Tech Upgrades:** full retail mat cost, then 80% if > 250eb |
| **DTD** | Fabrication downtime with **Tech HQ** (divide by 3, round up). Premium 1, Expensive 7, V. Expensive 14, Luxury/Super Luxury 28 per item tier |
| **Fab+Lab** | Fab materials + (DTD x **50eb**) |
| **DTD0** | Raw fabrication downtime (**no** Tech HQ). Premium 1, Expensive 7, V. Expensive 14, Luxury/Super Luxury 28 per item tier |
| **Fab+Lab0** | Fab materials + (DTD0 x **50eb**) |
| **Haggle** | Fixer acquisition at **20% off** book value (x0.80) |

*Biosystem (10,000eb) is separate from every body package - required before any FBC install.*
