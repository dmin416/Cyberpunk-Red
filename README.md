# Cyberpunk RED - Item Database

A comprehensive, interactive item lookup application for Cyberpunk RED tabletop RPG. Browse, search, sort, and view detailed information about weapons, armor, cyberware, gear, vehicles, and more!

## Features

- **Comprehensive Database**: 94+ items including weapons, armor, cyberware, gear, vehicles, ammunition, and more
- **Manufacturer-Specific Weapons**: Dai Lung, Militech, Federated Arms, Budget Arms and more
- **Advanced Cyberware**: Speedware (Kerenzikov, Sandevistan), Borgware, Chipware, and specialized options
- **Instant Preview on Hover**: Hover over any item to see a quick preview tooltip with stats and description
- **Advanced Search**: Search by name, description, tags, category, or subcategory
- **Smart Filtering**: Filter by category, subcategory, and availability
- **Multiple Sort Options**: Sort by name, cost, or category
- **Detailed Item Views**: Click any item to see complete stats and information in a full modal
- **Cyberpunk Aesthetic**: Dark theme with neon accents and glowing effects
- **Responsive Design**: Works on desktop, tablet, and mobile devices

## How to Use

1. Open `index.html` in your web browser (just double-click the file)
2. **Hover** over any item to see an instant preview tooltip
3. **Click** any item to view full detailed information in a modal
4. Use the search bar to find specific items
5. Apply filters to narrow down results by category, subcategory, or availability
6. Sort items by name or cost

## Item Categories

- **Weapons**: Pistols, SMGs, Shotguns, Assault Rifles, Sniper Rifles, Melee Weapons, Exotic Weapons, Grenades
- **Armor**: Head Armor, Body Armor
- **Cyberware**: Neuralware, Cybereyes, Cyberaudio, Cyberarms, Internal Cyberware, Fashionware
- **Gear**: Tech, Medical, General Equipment
- **Vehicles**: Motorcycles, Cars
- **Ammunition**: Various ammo types for different weapons

## Technologies Used

- Pure HTML, CSS, and JavaScript (no frameworks required)
- Responsive CSS Grid layout
- Custom cyberpunk-themed styling with neon effects
- Modal dialogs for detailed views
