# Inventory - Free-Range Kibble (Gig 6)
**Source of truth for objects.** If it exists on site, it has values here.
Pair with [Checks.md](Checks.md) · [Layout.md](Layout.md) · [Enemies.md](Enemies.md) · [Flavor.md](Flavor.md).
**Columns**
- **Sense** = glance / enter (no conclusion, no meta)
- **Check** = if they dig; payoff is facts only
- **SP / HP** = smash; aimed shots / Demo / heavy melee
- **Use** = E-stop, pick, dump, rescue - without needing to smash
Combat enemies keep full blocks in [Enemies.md](Enemies.md). Their install HP is duplicated here so you never hunt.
