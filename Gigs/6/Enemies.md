# Enemies - Free-Range Kibble (Gig 6)
**Default table:** **Cleric** (Medtech · shotgun · ROF 2) + **Weaver** (Tech · martial arts · ROF 2). Both **Hardened**. 
**Scale:** Use the **player-count tables** below for **1-4** Hardened edgerunners (add Solo/Netrunner/etc. as needed).
**All opposition is Hardened** (+2 attack/defend Skill Bases). Ceilings: **Mook 13** · **Lieutenant 15** · **Mini-Boss 17**.

**Durability = drones, not armored gangers.** Book drones are HP pools; personal drones often have **no SP**; Zhirafa combat drones top out around **SP 7-11** with **30-60 HP** (Observer 15 HP / Transporter 20 HP / Eagle SP7·30HP / Panther SP11·40HP / GRAF3 SP11·60HP). Line machines use that model - **not** LAJ SP12-14 + human HP.

| Tier | Combat HP | SP | Book anchor |
|------|----------:|---:|-------------|
| **Station mook** (Hook, Slicer, Prod, Clamp, Roller, optional Wands) | **35** | **0** | Between Transporter & Eagle - all damage to HP |
| **Station LT** (Frame, Arch, Cradle, Carrier) | **50** | **7** | Savannah Eagle plate |
| **Breakdown Saw** (Mini-Boss) | **60** | **7** | Eagle plate · GRAF3 HP - climax after attrition |
| **CUTTER terminal** (Tech obj, not a wave boss) | **40** | **0** | Smashable box / DV17 dump |

Hardened **+1 SP** does **not** apply - these are not wearing Armorjack. Structural smash values in [Layout.md](Layout.md) / [Inventory.md](Inventory.md) are for wrecking installs offline; in a fight, use the combat HP/SP above.

**Rate (IR2):** Hardened Mooks ~**1 per PC** · Hardened LTs ~**1 per 2 PCs**. One LT + N mooks per wave matches a 2-PC table; do not also give them ganger armor.
**Product line:** On the books - **scop slurry** + **wild animals** (Badlands game / ferals / pest harvest) for Continental Brands kibble/prepak. Tonight CUTTER-9 is running **people**. Tech **DV15** on the hardware → it was **designed for human bodies** under that cover (see Premise).
Machines. EMP / Microwaver = **Reset** (stunned 1 round) on Mooks (optional Elec/Sec **DV15** Action). LTs / CUTTER need **2** EMP hits or a hard kill.
Pair with [Premise.md](Premise.md) · [Layout.md](Layout.md) · [Checks.md](Checks.md).
**Engagement:** Stations are **tight**. Spawn mooks on the **small outer ceiling rails** above/beside the Crew. Keep the **central rail** clear until the Heavy Carrier arrives - do not call attention to it in read-aloud.

**Low-end Lethal:** One wave at a time. Give **1-2 rounds** Tech/Med breath after a clear if they earned it. Cook/Sear default to traps (not extra LTs). Carrier = reinforcement / climax weight - do not stack it with a full Roller swarm on round 1.

## Line progression (combat = reverse of product flow)

**Product:** Pens (people as livestock) → Sonic/Kill/Slice (**Saw**) → Tender → Portion → Cook → Sear → Package → Wrap → Dock.

**Combat:** Enter **Cold Dock**. Fight **upstream** to the **pens kill floor** (Sonic + Saw + living soft goods).

```
COLD DOCK / WRAP (entry · dead product only)
 → PACKAGE → SEAR → COOK → PORTION → TENDERIZE
 → PENS / KILL FLOOR (**Sonic + Kill Frame + Breakdown Saw + living soft goods**)
```

| Wave | Station | Harder enemy | Mook type |
|------|---------|--------------|-----------|
| **1** | Wrap / Dock | - (CUTTER = Tech dump) | Wrapper Rollers |
| **2** | Package | **Foam Cradle** (LT) | Packer Clamps |
| ** - ** | Sear, then Cook | - | traps / optional wands |
| ** - ** | Portion / smaller cuts | - | hazard / optional half-wave Slicers |
| **3** | Electrotenderize | **Tenderizer Arch** (LT) | Shock Prods |
| **4** | **Pens / Kill floor** | **Kill Frame** (LT) + **Breakdown Saw (Mini-Boss)** | Rail Hooks (+ Slicers at 3-4 PCs) |

**Sonic Array** lives in Wave 4 with the Saw - same bay as the pens. 
**CUTTER-9:** Dock mezz - Basic Tech / Elec/Sec **DV17** dump or smash (HP **40**, SP **0**). Killing the Saw also hard-stops the line.
**Bulkhead:** Dock→Pens shortcut. Tamper = **incinerate pens** (all living soft goods die).

## Heavy Carrier (central rail)
**When:** Wave 4 · **or** Perception **DV13** hears a long clank 1-2 rounds before · **or** pens combat drags past 4 rounds.
**When it arrives (read-aloud):** A long clank runs overhead. A cage drops onto the center rail. Restraint clamps open. A stencil on the cage reads PRIMARY PRODUCT.
**Level:** Hardened Lieutenant · **Rep:** 3
Use **Foam Cradle** STAT block as the base chassis (**HP 50 · SP 7**), then:

| Weapon | DMG | ROF | Notes |
|--------|-----|-----|-------|
| Restraint clamp | 3d6 | **2** | Grapple vs Athletics **15**; drag toward Wrap or Kill Frame |
| Body dump | 4d6 | 1 | Drops a grappled target prone (1/2 rounds) |
**Skill Bases:** Athletics **15** · Brawling **15** · Evasion **14** · Perception 12
**Notes:** Rides **central rail only** - can’t be rail-braked with the outer-rail trick (need Elec/Sec **DV17** on `MAINLINE_CARRIER` or Basic Tech **DV17** to spike the center track). Give the clank 1-2 rounds before it drops if you want a warning they can hear.
If pacing is already brutal, skip the Carrier.
