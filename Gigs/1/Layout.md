# Layout — Gig 1

Kabuki bar. Fight starts inside, spills to the street.

```
        STREET / CURB / PARKED CARS
                    │
              [6] Front door
                    │
    ┌───────────────┴───────────────┐
    │  [1] Bar rail                 │
    │  [2] Booth / tables           │  [5] Private room
    │  [3] Floor / dance-ish space  │◄── door
    │  [4] Back hall / toilets      │
    └───────────────────────────────┘
                    │
              [7] Alley (side exit)
```

---

## Zones

| # | Area | Notes |
|---|------|-------|
| **1** | Bar rail | Marshal + Weaver start here. Bottles = improvised; bartender ducks Round 1. |
| **2** | Booths / tables | Wave 1 Claws drinking. Flip tables = cover (SP 7–10 junk cover, GM fiat). |
| **3** | Open floor | Crossfire. Neon, sticky. |
| **4** | Back hall | Toilets, kegs, service door toward alley. |
| **5** | Private room | Wave 2. Soundproof-ish until gunfire; door opens when escalation triggers. |
| **6** | Street / curb | Spill zone. Cars, neon puddles, bystanders running. |
| **7** | Alley | Side exit; optional Claw escape or flanking. |

---

## Opening positions

| Who | Start |
|-----|-------|
| **Marshal, Weaver** | Zone 1 (drinking) |
| **Mooks ×3** | Zone 2 (booth / rail) |
| **LT Kaito** (+ optional Nori) | Zone 2 |
| **Civilians** | Everywhere — scatter Round 1; don't track unless needed |
| **Captain Ryo** (+ optional mooks) | Zone 5 until Wave 2 |

---

## Wave 2 trigger (pick one)

- Wave 1 cleared in **≤3 Rounds** with no PC Seriously Wounded
- Both LTs down and mooks folding while PCs are untouched
- Kaito radios / screams for the back room and someone answers
- GM fiat: fight feels too easy

**On trigger:** Private door opens. Ryo (and bodyguards) enter Zone 3 or hold the doorway (cover).

---

## Suggested flow

1. **Drink / tension** — Claws loud; Marshal tips over.
2. **Wave 1** — bar CQB; Weaver gadgets / Marshal chrome.
3. **Spill?** — survivors push for street (6) or alley (7).
4. **Wave 2** — if earned; private room empties into the wreckage.
5. **Extract** — no cops in time; Claw phones lighting up. Leave Kabuki or dig in.
