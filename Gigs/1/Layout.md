# Layout — Kabuki Kill Floor (Gig 1)

Kabuki bar. Fight starts inside, spills to the street. **Light rain** — hot, oily; **−1** Athletics / Drive Land Vehicle on slippery footing ([Premise.md](Premise.md#weather-session)).

```
        STREET / CURB / PARKED CARS
                    │
              [6] Front door
                    │
    ┌───────────────┴───────────────┐
    │  [1] Bar rail                 │
    │  [2] Booth / tables           │  [5] Private room
    │  [3] Floor / dance-ish space  │◄── door
    │  [4] Back hall / toilets      │
    └───────────────────────────────┘
                    │
              [7] Alley (side exit)
```

---

## Zones

| # | Area | Notes |
|---|------|-------|
| **1** | Bar rail | Marshal + Weaver start here. Bottles / *ochoko* / stools = improvised; bartender ducks Round 1. |
| **2** | Booths / tables | Wave 1 Claws drinking (vinyl booths, *tokkuri*+*ochoko*, beer cans, Smash cups, SCOP skewers / synthsquid). Flip tables = cover (SP 7–10). |
| **3** | Open floor | Crossfire. Neon, sticky + rain-tracked. Door spill = slippery (**−1** Athletics). |
| **4** | Back hall | Toilets, kegs, service door toward alley. Wet boots on tile. |
| **5** | Private room | Wave 2. Nicer chairs, Excellent Quality Alcohol ashtray money. Soundproof-ish until gunfire. |
| **6** | Street / curb | Spill zone. **2–3 CHOOH² Roadbikes**, Compact Groundcar, rainbow oil-slick puddles, bystanders running. **−1** Drive / Athletics on wet ferrocrete. |
| **7** | Alley | Side exit; optional Claw escape or flanking. |

## Reinforcements timer

**On first gunshot** (OOC):

1. Roll **3d6** → that **round number** = reinforcements arrive.
2. In-fiction: cousins on **CHOOH² Roadbikes** ~**30 seconds** after the first shot — same clock, narrate by round.

| Roll | Arrival |
|------|---------|
| **3–6** | Round 3–6 — fast |
| **7–12** | Round 7–12 — mid-fight |
| **13–18** | Round 13–18 — late; Wave 2 may beat them in |

**Where:** Zone **6** (street / curb) → push through door **6** or alley **7**. **2–4** Hardened Mooks (same block as Wave 1). Bikes still ticking from pre-fight.

Reinforcements are **independent** of Wave 2 (Ryo). Both can be live.

---

## Wave 2 trigger (pick one)

- Wave 1 cleared in **≤3 Rounds** with no PC Seriously Wounded
- Both LTs down and mooks folding while PCs are untouched
- Kaito radios / screams for the back room and someone answers
- GM fiat: fight feels too easy

**On trigger:** Private door opens. Ryo (and bodyguards) enter Zone 3 or hold the doorway (cover).

---

## Suggested flow

1. **Drink / tension** — Claws loud; Marshal tips over.
2. **Wave 1** — **3 mooks + 2 LTs**; roll **3d6** at first shot for reinforcements.
3. **Reinforcements?** — round = 3d6; street/bikes.
4. **Spill?** — survivors push for street (6) or alley (7).
5. **Wave 2** — if earned; Ryo from private room.
6. **Extract** — Claw phones lighting up.
7. **Breach** — **always** after combat; cusp check; [Flavor.md](Flavor.md#breach--after-combat).
