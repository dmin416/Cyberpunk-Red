# Layout — Until Morning (Gig 5)

Not a fight map. Investigation → approach → wreck puzzle. Zones are for movement, **time cost**, and who can help with rubble.

Pair with [Premise.md](Premise.md) · [Enemies.md](Enemies.md).

|----------|------|

## D — Wreck close-up (rubble puzzle)

```
         RIDGE WALL
    /////////////////
    //// RUBBLE //////
    //// ═══════ ////
    //// │ CAR │ ////  ← Hale + Juno; life signs DV15 Perception
    //// ═══════ ////
    //// bike scrap //
    /////////////////
         open scrub
         (Crew approach)
              │
    help arrives from S (dust cloud):
      Reef tow · Woodchipper hands · Kel GRAF3
```

**Reach rules:** Can’t pull them out until rubble shifts ([Premise.md](Premise.md) / [Enemies.md](Enemies.md)).

**Staging tips:**

- Call inventory (rope, chain, tools, explosives, Medtech kit) before they commit
- If they call help, paint the dust cloud as the clock’s last grain
- Stabilize **in place** in ridge shade before loading either body
- Optional partial shift: Medtech works one body while Tech finishes the other
