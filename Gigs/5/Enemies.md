# Enemies — Until Morning (Gig 5)

*No default combat roster. Opposition is the clock, environment, and injury states. Pair with [Premise.md](Premise.md).*

**Hardened rule:** N/A unless you add optional spice (below).

---

## Primary opposition — The Clock

| Fact | Detail |
|------|--------|
| **Win** | Find wreck · free both · stabilize both **before dawn** |
| **Fail** | One or both die from injuries / exposure / bad demo |
| **Crew lethality** | Low (sun fatigue, vehicle damage, failed explosives splash) |
| **Victim lethality** | High if slow, noisy with explosives, or move unstabilized |

| Phase | Push it by… |
|-------|-------------|
| Morning → Heat | Longer canvass; failed locks; arguing over plans |
| Amber | Reef Agent ping: *“sun’s dropping. car still gone.”* |
| Night | Headlamp circles; wind; optional scav |
| Last hours | When they crack the car: shallow breath, sticky heat, “they’re not going to make another day” |

## Environment

### Midday Badlands (walk-in)

| Check | DV | Fail |
|-------|-----|------|
| Endurance (walking under sun) | 13 | Weary: **−2** all Checks until a full night’s sleep |

Dust storm was **yesterday** — tracks buried; +0 DV if they somehow search the same day as the crash (they won’t).

### Find the wreck

| Method | DV | Time |
|--------|-----|------|
| Tracking | 15 | — |
| Perception (buried tracks) | 17 | — |
| Tactics → spiral search | 15 | ~30 min to car |
| Random search | — | ~1 additional hour |

### Rubble (Hazardous obstacle)

| Approach | Requirement / DV | Fail |
|----------|------------------|------|
| Hands | 2+ characters **BODY 10+** together | Can’t shift |
| Vehicle tow | Drive **17** (Heavy Chassis) / **24** (no upgrade) | Rubble moves; vehicle takes **3d6** damage |
| Explosives | ≥2 grenades + Demolitions **24** | **Both NPCs die** |
| Levers / fulcrums | Right gear + Basic Tech **21** | No shift |
| Call help | Social **13–17** (Reef, Woodchipper, Kel Sato + GRAF3) | Delay burns clock |
| Partial shift (optional) | GM fiat on a near-success | One victim reachable; other still pinned |

## Victims (not enemies)

Treat as **stabilization puzzles**, not combatants. Both at **0 HP**, **unstabilized**, cannot fight or walk out.

### Marcus Hale — PI

| | |
|--|--|
| **Role** | Private investigator; Nomad-circuit freelance |
| **HP** | 0 (Dying — RAW timing as you prefer) |
| **Need** | Stabilize before move; Cracked Skull hates bumps |

**Critical Injuries:** Broken Ribs · Cracked Skull · Whiplash

**Look:** Sepia trench aesthetic, scuffed “armor fantasy,” blood on faux leather. Agent cracked but maybe salvageable later.

**If he lives:** Owes one free dig / skip-trace. Soft spot for mystery novels and Woodchipper’s people.

### Juno Vale — Rockerboy

| | |
|--|--|
| **Role** | Between-gigs musician; NCU music grad |
| **HP** | 0 |
| **Need** | Stabilize before move; Spinal / Broken Leg / Foreign Object make rough handling ugly |

**Critical Injuries:** Broken Leg · Foreign Object · Spinal Injury · Whiplash

**Look:** Colorful thrift under dust; camping grit; bike frame twisted in the rubble scatter.

**If they live:** Recruitable musician contact; will play Last Stop for free once. Depression doesn’t vanish because you saved them — treat gently.

## Medical reference (field)

| Step | Check (sample) | Notes |
|------|----------------|-------|
| Stabilize Hale | First Aid / Paramedic **DV15** | Dust, wreck angle, bad light |
| Stabilize Juno | First Aid / Paramedic **DV15** | Same; Spinal — no rolling them like a carpet |
| Foreign Object (move-safe) | Paramedic / Medicine **DV17** | Or leave it and accept worse transport risk |
| Package Spinal | Paramedic **DV15** | Board / improvised brace |
| Move unstabilized | — | They’re dead or dying on arrival — don’t allow “we’ll fix it in the truck” without a hard ruling |

Two Medtech actions can run in parallel if two characters work; one Medtech splits time while Tech finishes rubble.

## Optional spice (only if the table is bored)

Do **not** add by default for a Tech/Medtech duo.

| Spice | Use |
|-------|-----|
| Distant Raffen gunfire | Atmosphere — original job beat; no contact |
| **1 scavenger Mook** after a long loud demo | Wants scrap, not a duel — bribe / scare / fight |
| Dust picks up again | +1 DV outdoor Checks; clock pressure |
| Night chill | Endurance **DV13** if they wait without shelter |

If you fight a scav, use a corebook **Boosterganger** or **Road Ganger** Mook — non-Hardened.

| Scav Mook (quick) | |
|-------------------|--|
| HP 20 · SP 7 · Combat# ~10 | Heavy Pistol 3d6 or crowbar 2d6 |
| Motive | Metrocar scrap / bike parts |
| Flee | If outnumbered or paid ≥100eb scrap value |
