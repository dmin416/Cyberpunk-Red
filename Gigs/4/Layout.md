# Layout — Gig 4

Kabuki side street. Fight stays **outside** — sushi stall, curb, traffic. No bar interior.

```
    [8] Alley mouth          [7] Ramen / shutter shops
              │                        │
    ┌─────────┴────────────────────────┴─────────┐
    │  [5] Parked bikes · compact groundcar      │
    │  [4] TRAFFIC LANES (semi-active) ◄─────────┤── flow →
    │  [3] Sidewalk / curb fight zone            │
    │  [2] Sushi stall (counter · steam · gas)   │
    │  [1] Crew tables / standing eat            │
    └────────────────────────────────────────────┘
              │
         [6] Cross street (more traffic · witnesses)
```

## Zones

| # | Area | Notes |
|---|------|-------|
| **1** | Crew start | Low tables or standing at the stall. Exec's purse / agent on the counter; Medtech kit under the bench. |
| **2** | Sushi stall | **Cover SP 7** (counter, awning frame). Propane / steam vent — Weaver can weaponize. Vendor ducks Round 1. |
| **3** | Sidewalk / curb | Main fight. Neon signs, vending machine (SP 10), news kiosk. Wave 1 Claws here. |
| **4** | Traffic lanes | **Semi-active.** See [Traffic](#semi-active-traffic). Moving cover, collision risk, blocked shots. |
| **5** | Parked vehicles | **2–3 CHOOH² Roadbikes** (Claw colors), one **Compact Groundcar** (SP 13 body if crouched behind). Wave 2 bikes may slot in here. |
| **6** | Cross street | Bystanders, distant sirens (never in time). Optional second car horn swarm. |
| **7** | Shop line | Shutters half-down; civilians pile in. |
| **8** | Alley | Claw escape or Nori's late entrance if you want flanking drama. |

## Semi-active traffic

Roll or pick each **Round** after Round 1 (or when someone enters lane 4):

| d6 | Event |
|----|-------|
| **1–2** | **Clear lane** — no effect this round |
| **3–4** | **Dense pack** — crossing lane 4 costs **2 MOVE**; attacks across traffic **−2** (glare, motion, panic) |
| **5** | **Hard brake** — one **Compact Groundcar** stops in zone 4 (**cover SP 13**); driver flees |
| **6** | **Near miss** — anyone in lane 4 without cover: **DV11 Evasion** or **2d6** vehicle damage (ignores armor if failed by 5+) |

**Optional:** When Wave 2 arrives, a bike **lays rubber through lane 4** — reinforcements get +2 MOVE first round they enter.

**Medtech angle:** Crushed-by-car criticals are on the table if someone eats a **6** in the open.

## Wave timing (GM)

| Clock | Trigger |
|-------|---------|
| **Round 1** | Wave 1 — **3 mooks** engage |
| **Round X** | **1d6** rolled at fight start — Wave 2 (**1 mook + 2 LTs**) |
| **Round X+1** | Wave 3 — **Nori** (Mini-Boss) **only if crew still fighting**. **Flee = skip Wave 3 entirely.** Delay to X+2 if PCs still engaged but hurting |

**Audio cue for Wave 2:** twin bike engines + someone yelling *"Nori said kill the chrome!"*

**Audio cue for Wave 3:** shotgun rack + Nori on foot: *"You don't get to embarrass us on my block."*

## Suggested flow

1. **Eat / tension** — Claws cruise past, double back, dismount.
2. **Wave 1** — sidewalk CQB; stall steam, vending cover.
3. **Traffic** — lane 4 matters; Solo holds the curb, Exec don't get isolated.
4. **Wave 2 (Round X)** — bikes slot in; LTs with SMG/AR.
5. **Wave 3 (Round X+1)** — Nori with shotgun; fight peaks.
6. **Extract** — strip bikes, vanish before Claw phones light up the district.
